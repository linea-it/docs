# Repositorio de Datos

El acervo de datos de LIneA reúne catálogos astronómicos públicos, imágenes y productos de photo-z. La lista completa de los datos disponibles, junto con información detallada e instrucciones de acceso, está disponible en la página dedicada a los datos de LIneA en [**data.linea.org.br**](https://data.linea.org.br/).
