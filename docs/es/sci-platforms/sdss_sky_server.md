El *SDSS Sky Server* es un portal en línea desarrollado por el *Sloan Digital Sky Survey* (SDSS) que proporciona acceso público a una vasta colección de datos astronómicos recopilados por sus relevamientos. Ofrece herramientas interactivas para visualizar imágenes del cielo, explorar espectros y consultar catálogos de objetos celestes, como estrellas, galaxias y cuásares. Con una interfaz intuitiva y recursos que atienden tanto a entusiastas como a investigadores profesionales, el *Sky Server* también ofrece formas de acceder a los datos mediante consultas SQL, facilitando análisis personalizados directamente sobre la base de datos de SDSS.

LIneA aloja un espejo del *SDSS Sky Server* ([**skyserver.linea.org.br**](http://skyserver.linea.org.br/sciserver/)), donde proporciona acceso a los datos del *release* DR17.
